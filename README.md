# NAO README.md

## Introduction
This is the STARI labs collection of programs for the NAOs.

## Instructions:
1. start running a model
```
ollama run [model_name:num_parameters]
```
2. ssh into the NAO
```
ssh nao@nao.local
```
3. run the python script in the scripts folder
```
cd scripts
python sendrequestLLM.py
```
